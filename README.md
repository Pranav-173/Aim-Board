# Aim Board

Aim board is a clean and interactive goal-tracking web application that helps users visually manage their personal, academic, career, and health goals.  
It combines a **vision board**, **timeline**, and **analytics dashboard** to make goal-setting simple, motivating, and intuitive.

---

## Features :

- **Vision Board**
  - Card-based layout to visualize goals
  - Progress bars with percentage tracking
  - Overdue goals highlighted automatically

- **Timeline View**
  - Chronological view of all goals
  - Easy tracking of upcoming and past deadlines

- **Analytics**
  - Visual bar chart showing goal progress
  - Quick overview of overall productivity

- **Sorting Options**
  - Sort goals by:
    - Date
    - Progress
    - Category

- **Goal Management**
  - Add new goals with category, date, quote, and progress
  - Remove goals with confirmation
  - Data persists using browser Local Storage

- **Onboarding Experience**
  - Friendly welcome message shown on first visit

- **Responsive Design**
  - Works smoothly on desktop, tablet, and mobile devices

---

## Technologies Used

- **HTML5** – structure  
- **CSS3** – styling and responsive layout  
- **JavaScript** – application logic  
- **LocalStorage API** – data persistence  

(No external libraries or frameworks used)
